## HOW TO RECORD (Step-by-Step)

### For Demo Video:

1. **Set up** (5 min)
   - Open browser, navigate to localhost:3000
   - Test all functionality
   - Have code ready to paste

2. **Record** (10 min)
   - Use ScreenFlow (Mac) or OBS (all platforms)
   - Record at 1080p
   - Speak clearly, pause between sections
   - Do 2-3 takes

3. **Edit** (20 min)
   - Add intro title card (3 sec)
   - Cut out mistakes
   - Add captions for key points
   - Add background music (low volume)
   - Export as MP4

4. **Upload** (5 min)
   - Upload to YouTube (unlisted)
   - Get shareable link
   - Test link works

### For Pitch Video:

1. **Write script** (5 min)
   - Use the script above

2. **Practice** (10 min)
   - Read it out loud 2-3 times
   - Get timing right
   - Feel natural (not robotic)

3. **Record** (15 min)
   - Record talking head (on camera)
   - Record screen demo (if using hybrid)
   - Pick best takes

4. **Edit** (30 min)
   - Combine clips
   - Add transitions
   - Add music
   - Add captions
   - Color correct
   - Export MP4

5. **Share** (2 min)
   - Upload to YouTube
   - Share link
